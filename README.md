# 任务管理系统

一个简洁高效的Web任务管理系统，帮助您跟踪行动项、管理任务进度。

## 功能特点

- ✅ **快速录入**：轻松添加任务内容、责任人和计划完成时间
- 🔄 **自动状态**：根据当前时间自动判断任务状态（进行中/已延期/已完成）
- 📊 **数据看板**：实时展示任务统计和可视化图表
- ⏰ **智能提醒**：7天内到期任务高亮显示
- 👥 **责任人统计**：按责任人维度查看任务分布
- 💾 **本地存储**：数据存储在浏览器本地，无需服务器

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 填写任务信息并提交
3. 在看板上查看统计信息
4. 使用筛选器查看不同状态的任务

## 任务状态说明

| 状态 | 说明 | 颜色 |
|------|------|------|
| Ongoing（进行中） | 当前时间在计划完成时间之前 | 蓝色 |
| Delay（已延期） | 当前时间已超过计划完成时间 | 红色 |
| Close（已完成） | 任务已手动完成 | 绿色 |

## 提醒规则

- **即将到期**：7天内到期的进行中任务显示黄色警告
- **已延期**：超过deadline的未完成任务显示红色警告
- **高亮警告**：当7天内到期任务超过3个时，指标卡片变红色

## 技术栈

- HTML5 + CSS3 + JavaScript（原生）
- LocalStorage（数据持久化）
- Chart.js（数据可视化）

## 数据存储

所有数据存储在浏览器的 LocalStorage 中，可以安全地关闭浏览器，数据不会丢失。

如需清除数据，在浏览器控制台执行：
```javascript
localStorage.removeItem('taskManagerData');
```
